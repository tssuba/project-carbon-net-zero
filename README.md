# Carbon-Net-Zero

## Docker Instructions:

- Command:
  - docker-compose up -d
- Command to see container logs:
  - docker-compose logs -f <container_name or container ID> eg: backend, frontend, proxy

#

## To access the application:

- backend server:
  - url: http://api.localhost
  - For api docs
    - url: http://api.localhost/docs
- frontend client:
  - url: http://localhost
